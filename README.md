# PAW-FetchFromDBDynamicValue

A [Paw](https://paw.cloud/) extension to fetch value from a database (currently limited to [mysql](https://www.mysql.com/)).

#### Prerequisites

```
nvm install
nvm use
npm install
```

#### Build

```
npm run build
```

#### Install

```
make install
```

#### Test

```
npm test
```
